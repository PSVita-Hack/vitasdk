# vitasdk
VITASDK for OS X

add some dependencies and some stubs

## Install

```shell
cd /usr/local
sudo git clone https://github.com/Yggdrasill-7C9/vitasdk.git
echo "export VITASDK=/usr/local/vitasdk" >> ~/.bashrc
```
